# Mochila de Viagem

Projeto desenvolvido durante o curso _JavaScript na Web: Armazenando dados no navegador_, da **Alura**.

## Sobre o projeto

Este projeto consiste em criar uma lista, adicionando um ítem novo toda vez que o usuário preencher e enviar o formulário. É também um exercício de **CRUD** (Create, Retrieve, Update, Delete) e de armazenamento no **Local Storage** do navegador.
